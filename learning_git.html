Learning Git

Git is a powerful version control system that helps developers track changes in their code.

Basic Git Commands

  • git init - Initializes a new Git repository.
  • git clone <repo-url> - Clones a remote repository.
  • git status - Checks the status of your repository.
  • git add . - Stages all changes for commit.
  • git commit -m "Your message" - Commits staged changes.
  • git push origin branch-name - Pushes your changes to the remote repository.

Why Use Git?

Git makes collaboration easy by allowing multiple developers to work on the same project without conflicts. It also helps track code history and revert changes if needed.
